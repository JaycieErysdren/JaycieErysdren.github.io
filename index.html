____....----==== repositories ====----....____

B-ReX :: an attempt to wrap up the opengl version of brender with the 1995 sdk version, allowing for a rendering engine compatible between dos and modern platforms.

DOSLIB :: an early idea for a compatibility layer between the open watcom v2 dos compiler and the djgpp dos compiler. they share many of the same features, but lack any kind of shared syntax.

Formats :: a collection of kaitaistruct definitions for various obscure binary formats, mostly related to videogame assets and media. probably my most useful repository.

Liberator :: an electron prototype for a gui asset viewer, focusing on the file formats enumerated in the previously listed repository.

MarcLaidlaw-Epistle3 :: for some reason in 2017 i thought it would be a great idea to mirror marc laidlaw's "epistle 3" blog post on github, with the idea that i could crowdsource translations and syntax updates.

Portal2D-SDL :: my workspace for toying with the portal2d engine, a 2.5d portal-based dos rendering engine created in 1999 by derek john evans.

RexEngine :: a multi-purpose engine written in c, designed for making gui applications and games

SaturnQuakePC :: a semi-abandoned port of the sega saturn version of quake backwards to pc.

Stanley96 :: a semi-abandoned port of the stanley parable to the fteqw quake engine sourceport.
